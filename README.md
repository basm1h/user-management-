# user-management-
A database-driven web application for managing user information, developed using Flask (Python) and SQLite. The system supports full CRUD operations and features a responsive, user-friendly interface built with Bootstrap. It demonstrates key concepts of relational database design, data normalization, and dynamic web development.
 # 💜 User Management System

## 📌 Project Description
This project is a database-driven web application developed using **Flask (Python)** and **SQLite**.  
It allows users to manage personal data through a simple and user-friendly interface.

The system supports full **CRUD operations**:
- Create (Add user)
- Read (View users)
- Update (Edit user details)
- Delete (Remove user)

---

## 🚀 Features
- Add new users with full details
- Edit existing user information
- Delete users from the system
- Display all users in a clean dashboard
- Responsive and modern UI using Bootstrap
- Data stored using SQLite database

---

## 🧱 Technologies Used
- Python (Flask)
- SQLite (Database)
- HTML / CSS
- Bootstrap (UI Design)
